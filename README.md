# QA Portfolio Project – Test Design App

## 📌 Description

This project is a simple fullstack application designed to demonstrate QA skills, including:

* test design techniques explanation
* API testing
* UI testing

The application displays test design topics in an accordion format.

---

## 🧩 Tech Stack

* Backend: FastAPI (Python)
* Frontend: HTML + JavaScript
* Hosting:

  * Backend: Render
  * Frontend: GitHub Pages

Note: Backend is hosted on a free Render instance, so the first request may be slower due to cold start.

## ⚙️ Features

* Fetch topics from API
* Display topics in accordion UI
